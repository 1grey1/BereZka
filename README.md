# Разработка MVP приложения для обмена мгновенными сообщениями
Полу-децентрализованный анонимный мессенджер для безопасной текстовой переписки без единой точки отказа.
## Команда, распределение

- **Анатолий**: frontend, интеграция с backend, логика экранов.
- **Богдан**: frontend, дизайн UI, Figma-макеты.
- **Сергей**: backend, P2P-сеть (libp2p, DHT, NAT traversal).
- **Ярослав**: backend, криптография, хранение (SQLite + SQLCipher).

## Цель проекта

Разработать прототип мобильного P2P-мессенджера для обмена текстовыми сообщениями с E2EE без центрального сервера.

## Технологии

| Компонент | Технологии |
| :-- | :-- |
| Frontend | Kotlin (Android), Coroutines, XML/Compose |
| Backend | Rust, libp2p (Kademlia DHT, Gossipsub, NAT) |
| БД | SQLite + SQLCipher (AES-256) |
| Инструменты | GitHub, Figma, Android Studio, Cargo |
